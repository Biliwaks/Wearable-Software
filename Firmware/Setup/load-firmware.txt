To load the firmware on the bbb, use an FTP client like Filezilla and transfer the folder Firmware.

Navigate to the src folder and type in following command:
>> python3 Main.py


Use following command to get all GPIO's:
>> mraa-gpio list


Config-pin is used to set pin mode:
https://github.com/beagleboard/bb.org-overlays/tree/master/tools/beaglebone-universal-io
